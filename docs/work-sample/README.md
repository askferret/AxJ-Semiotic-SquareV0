# EPIC 2026 Work Sample — Screenshots and Materials

Use this folder for **Work Sample** materials for the EPIC 2026 Prototype/MVP submission: screenshots, and optionally a demo video or link to the live prototype.

---

## Recommended screenshots (for the application committee)

Capture the following in order. Save them in this folder with the suggested filenames so reviewers get a clear sense of the flow.

| # | Filename | What to capture |
|---|----------|-----------------|
| 1 | `01-landing-empty.png` | **Landing / input screen** — Empty state: header “Semiotic Square — Source Mapping,” subheading, mode toggle (Quick analyze / Guided worksheet), empty text area, model selector, Generate button (disabled), “Try Sample” / “Sample Mode” link, and the short “Enter a story pitch…” empty-state message. Shows the entry point. |
| 2 | `02-input-with-pitch.png` | **Input with text** (optional but helpful) — Same as above but with a short story pitch or field notes in the text area (e.g. 2–3 sentences about a beat or conflict). Shows what users paste before generating. |
| 3 | `03-results-full-page.png` | **Full results after Sample or Generate** — After clicking “Try Sample” or “Generate”: title of the map (e.g. “Belonging vs. Exclusion”), axis, the **8-slot diagram** (corners 1–4 and edges 1-2, 2-3, 3-4, 1-4) with labels visible, and the “Who to look for” section with at least one set of source prompts. Full-page scroll is ideal so the structure is obvious. |
| 4 | `04-diagram-and-actions.png` | **Diagram + export actions** — Viewport or scroll so the **8-slot grid** and the **Copy All** / **Export PDF** buttons are visible. Shows that results are editable and exportable. |
| 5 | `05-guided-mode.png` | **Guided worksheet mode** (optional) — Switch to “Guided worksheet,” load a sample or generate once, then capture the step indicator (e.g. “Step 2 of 4”) and the oppositions confirmation (or step 3/4). Shows the alternative, step-by-step flow. |

**Minimum for a strong work sample:** 1, 3, and 4. Add 2 and 5 if you have space.

---

## Optional additions

- **Demo video** (e.g. 2–3 min): Screen recording showing paste → Generate (or Try Sample) → scroll through 8-slot diagram and “Who to look for” → Copy All or Export PDF. Name it e.g. `demo-semiotic-square-source-mapping.mp4` and add to this folder or link in the submission.
- **Live prototype link:** If you deploy (e.g. Vercel), include the URL in the submission; add password/credentials in the application form if access is restricted for reviewers.
